# Limitations and Ethics Assessment

## Limitations

- SMS export data may miss transactions performed through other channels.
- Transaction categories are inferred from message text and may misclassify edge cases.
- Lack of demographic attributes limits fairness and bias assessment.

## Ethical Considerations

- Data should remain anonymized and stored securely.
- Avoid using the model for high-stakes decisions without additional validation.
- Ensure informed consent for any data collection and use.

## Future Work

- Integrate demographic data for richer segmentation.
- Validate results on a larger, more diverse sample.
- Incorporate additional temporal features for better seasonality capture.
