# EDA Summary

Key patterns observed:

- Activity levels are imbalanced with a small number of high-activity users.
- Transaction volume increases sharply with activity level.
- Failed attempts occur more frequently for high-activity users.
- Weekend activity rises for medium and high activity segments.
