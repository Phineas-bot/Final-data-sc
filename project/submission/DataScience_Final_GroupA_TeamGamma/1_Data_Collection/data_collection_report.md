# Data Collection Report

## Sampling Strategy
Participants were selected to ensure diversity by age range, profession, and geographic zone. Each group member collected at least 10 participants with informed consent, targeting a minimum of 50 participants overall.

## Data Sources
- SMS export histories from MobileMoney and OrangeMoney.
- Questionnaire responses for demographic attributes.

## Data Quality Controls
- Assigned anonymized user IDs to all participants.
- Removed personal identifiers before analysis.
- Checked for missing or inconsistent records during ingestion.

## Challenges
- Some participants had incomplete SMS histories.
- SMS formats varied across providers; normalization was required.

## Ethical Considerations
- Participation was voluntary with the right to withdraw.
- Data stored securely and used only for academic purposes.
