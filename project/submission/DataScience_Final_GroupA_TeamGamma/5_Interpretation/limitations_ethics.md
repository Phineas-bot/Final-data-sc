# Limitations and Ethics

Limitations:
- User-level sample size is small, limiting generalization.
- SMS parsing can misclassify edge cases.
- Demographic variables are not fully integrated in modeling.

Ethics:
- Data is anonymized and stored securely.
- Results should not be used for high-stakes credit decisions.

Future Work:
- Expand participant sample size.
- Integrate demographic features for fairness analysis.
- Add temporal sequence modeling for improved prediction.
